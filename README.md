# Guia Digital Comunitário - Rio de Janeiro

## 📝 Descrição do Projeto
O **Guia Digital Comunitário** é um projeto de extensão desenvolvido para a disciplina de **Desenvolvimento Frontend** da **Universidade UNISUAM**. O site funciona como um hub informativo para conectar moradores do Rio de Janeiro a serviços sociais essenciais, oportunidades e instituições de apoio.

## 🎯 Objetivo Social
A finalidade deste projeto é fortalecer a relação entre tecnologia e comunidade, oferecendo um recurso digital de interesse público que facilite a busca por auxílio em áreas como segurança alimentar, educação e assistência social básica.

## 🚀 Tecnologias Utilizadas
Para o desenvolvimento deste projeto, foram aplicadas as seguintes tecnologias e ferramentas:
* **HTML5**: Estruturação semântica do conteúdo.
* **CSS3**: Estilização visual com foco em responsividade (Mobile First).
* **JavaScript**: Interações simples com o usuário.
* **VSCode**: Editor de código principal.
* **Git & GitHub**: Versionamento de código e hospedagem do repositório.
* **GitHub Pages**: Publicação e disponibilização do site online.

## 📱 Funcionalidades
* **Página Inicial**: Apresentação clara do objetivo do projeto.
* **Cards Informativos**: Seção dedicada a instituições parceiras como a *Ação da Cidadania*, *Redes da Maré* e o *CRAS*.
* **Interface Responsiva**: Layout adaptado para telas de computadores e dispositivos móveis.
* **Links Diretos**: Botões interativos que direcionam o usuário para os canais oficiais das instituições.
